# 说明
项目分为2个服务，一个是本地化跑任务的，一个是放在服务器上用于通信的。

目前仅接入的钉钉机器人，需要进入企业开发后台新建机器人应用。

# 服务端
负责钉钉机器人的运行，发送消息。将机器人添加如内部群后，可以@机器人，请求一个登录链接，或者 请求统计数据

# 任务端
该端用于实际执行任务，放在了本地环境，这样可以确保非服务器IP在学习
